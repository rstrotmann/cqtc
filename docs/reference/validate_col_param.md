# Validate character parameter

Validate character parameter

## Usage

``` r
validate_col_param(
  value,
  df = NULL,
  allow_null = FALSE,
  allow_multiple = FALSE
)
```

## Arguments

- value:

  The parameter value.

- df:

  A data frame the column names of which must contain the value(s), if
  not NULL.

- allow_null:

  Allow NULL values, as logical.

- allow_multiple:

  Allow character vectors, as logical.

## Value

Nothing.
