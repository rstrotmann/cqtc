# Generate the XXH128 hash of a cqtc object

Generate the XXH128 hash of a cqtc object

## Usage

``` r
# S3 method for class 'cqtc'
hash(obj)
```

## Arguments

- obj:

  A catc object.

## Value

The XXH128 hash of the catc object as character.

## Examples

``` r
hash(dofetilide_cqtc)
#> [1] "1bdc4638d07dea941ca833398945214a"
```
