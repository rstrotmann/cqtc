# Time course plot for QT parameter and drug concentration

Time course plot for QT parameter and drug concentration

## Usage

``` r
cqtc_time_course_plot(obj, param = "QTCF", title = "")
```

## Arguments

- obj:

  A cqtc object.

- param:

  The parameter to be plotted, as character.

- title:

  The plot title.

## Value

A ggplot object.
