# Plot HR over time

Plot HR over time

## Usage

``` r
hr_by_time_plot(
  obj,
  param = "HR",
  group = NULL,
  title = "",
  size = 0.5,
  alpha = 1,
  lwd = 0.6
)
```

## Arguments

- obj:

  A cqtc object.

- param:

  The parameter to plot.

- group:

  The grouping variable.

- title:

  The plot title.

- size:

  Point size.

- alpha:

  The alpha value for points.

- lwd:

  The line width.

## Value

A ggplot2 object.
